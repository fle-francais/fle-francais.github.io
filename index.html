Le passé composé Grammaire

Formation : AVOIR/ÊTRE + participe passé

InfinitifPassé composéExemple
mangerj'ai mangéJ'ai mangé une pizza.
partirje suis parti(e)Elle est partie à 8h.
finirj'ai finiTu as fini le livre ?

💡 Les verbes de mouvement (aller, venir, partir…) utilisent ÊTRE.

Les prépositions de lieu Grammaire

PrépositionExemple
sur (on)Le livre est sur la table.
sous (under)Le chat est sous le lit.
dans (in)Les clés sont dans mon sac.
devant (in front of)Il attend devant la porte.
derrière (behind)Le jardin est derrière la maison.
à côté de (next to)La banque est à côté du café.

L'imparfait Grammaire

Formation : radical nous (présent) + terminaisons

PronomTerminaisonParler → parlais
Je / Tu-aisje parlais, tu parlais
Il / Elle-aitil parlait
Nous-ionsnous parlions
Vous-iezvous parliez
Ils / Elles-aientils parlaient

💡 Utilisation : description dans le passé, habitude passée, contexte d'une action.

Imparfait ou passé composé ? QCM

« Quand je ___ petit, j'habitais à Paris. »

A étais
B ai été
C suis
D fut

Mettre à l'imparfait Texte à trous

1. Avant, nous (manger) ensemble chaque soir.
2. Elle (lire) un roman quand je suis arrivé.
3. Vous (habiter) où à l'époque ?
