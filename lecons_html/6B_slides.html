Exercices 1/2
✏️ Exercice 1 — Exprimez vos goûts 📝 Grammaire
Complétez avec le bon verbe (adorer, aimer, ne pas aimer, détester) : · 用正确的动词填空（adorer, aimer, ne pas aimer, détester）
1. La musique classique… Vous la trouvez belle ? → J' la musique classique.
2. Le bruit, c'est insupportable pour vous ? → Je le bruit.
3. Le rap vous laisse indifférent(e) ? → Je n' pas trop le rap.
4. Le jazz, c'est votre passion ? → J' le jazz.
✏️ Exercice 2 — Quel verbe exprime ce sentiment ? 📝 Grammaire
Choisissez le bon verbe selon l'intensité du sentiment : · 根据情感强度选择正确的动词
1. 😍 Cette chanson est magnifique, je l'écoute 10 fois par jour ! → J'___ 😍 这首歌太美了，我每天听10遍！→ 我___
2. 😐 Ce style musical, ce n'est pas vraiment pour moi. → Je n'___ pas trop. 😐 这种音乐风格不太适合我。→ 我不太___。
3. 😤 Cette musique m'énerve profondément. → Je ___ 😤 这种音乐让我非常烦躁。→ 我___
4. 🙂 J'écoute parfois ce groupe, c'est plutôt bien. → J'___ bien. 🙂 我有时听这个乐队，还挺好的。→ 我___。
5. 😊 Ce concert était correct, sans plus. → J'___ un peu. 😊 这场音乐会还可以，不过不是特别好。→ 我有点___。
✏️ Exercice 3 — Formes négative et interrogative 📝 Grammaire
Transformez les phrases : · 改写句子

2. Tu adores la musique classique. → Forme négative :
3. Elle préfère le jazz. → Forme interrogative (est-ce que) :
4. Vous écoutez souvent de la musique. → Forme interrogative (inversion) :
✏️ Exercice 4 — Vocabulaire : les goûts et la musique 📚 Lexique
Classez les verbes du moins positif au plus positif : détester / apprécier / adorer / ne pas aimer / aimer Puis complétez : 1. J'___ le rock, c'est ma passion ! · 按从最不积极到最积极的顺序排列动词
