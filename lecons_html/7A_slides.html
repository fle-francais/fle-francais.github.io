🎵 Phonétique · 语音
§ Phonétique — Le [ʃ] vs le [ʒ] Règle [ʃ] : ch, comme dans « chat ». [ʒ] : j, ge, comme dans « jeune » ou « age ». Exemples chercher [ʃ] — je [ʒ] — chemise [ʃ] — rouge [ʒ] Exercice Distinguez : chaque / Jacques — short / Georges — chanson / jambon — chaussure / jaloux.
📖 Point de langue · 语法要点
§§ Ce devant masculin. Cet devant masculin + voyelle/h muet. Cette devant féminin. Ces devant pluriel.
• Cette robe est magnifique. / Ces chaussures sont confortables.
• On peut ajouter -ci / -là : cette veste-ci (ici) / cette veste-là (là-bas).
• Forme négative : Ce pantalon n'est pas trop court. / Ces chaussures ne sont
• Forme interrogative : Est-ce que cette robe est en soldes ? / Ces chaussures

✏️ Exercice 1 — Ce, cet, cette ou ces ? 📝 Grammaire
1. pantalon te va très bien.
2. chaussures sont à 40€.
3. imperméable est imperméable au vent.
4. robe est trop courte.
5. vêtements sont en soldes.
✏️ Exercice 2 — Jeu de rôle : en boutique 📝 Grammaire
À deux : un vendeur, un client. Le client cherche : un pull pour l'hiver. Le vendeur propose différents modèles avec les démonstratifs : Ce pull…, Cette couleur…, Ces manches… Négociez la taille, la couleur, le prix. · 两人一组：一名售货员，一名顾客
