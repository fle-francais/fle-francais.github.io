📖 SAVOIR vs CONNAÎTRE · 语法要点
Règle
SAVOIR vs CONNAÎTRE
VerbeUsageExemple
SAVOIRsavoir faire qqch / un faitJe sais parler chinois. / Il sait que tu pars.
CONNAÎTREêtre familier de qqn/qqchJe connais Paris. / Tu connais Marie ?
• Conjugaison : je sais / tu sais / il sait / nous savons / vous savez / ils savent
• je connais / tu connais / il connaît / nous connaissons / vous connaissez / ils connaissent
Règle
Les pronoms relatifs QUI et QUE
PronomRôleExemple
QUIsujetC’est l’ami qui parle français.
QUE / QU’objetC’est le film que j’ai vu hier.
• QUI + verbe : La personne qui travaille ici est sympa.
• QUE + sujet + verbe : Le livre que tu m’as prêté est génial.
