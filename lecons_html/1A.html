Elle est italienne.

📖 Point de langue — Le verbe ÊTRE au présent
§§ Le verbe être est irrégulier. On l'utilise pour exprimer l'identité, la nationalité et la profession.
SingulierPluriel
je suisnous sommes
tu esvous êtes
il / elle estils / elles sont
• Nous sommes dans le même cours de français.
• Forme négative : Je ne suis pas professeur. / Tu n'es pas espagnole.
• Forme interrogative : Est-ce que vous êtes étudiant(e) ? / Vous êtes d'où ?
✏️ Exercice 1 — Conjuguez le verbe ÊTRE
1. Je français(e).
2. Il étudiant en médecine.
3. Nous dans la même classe.
4. Vous professeur de français ?
5. Elles espagnoles.
✏️ Exercice 2 — Présentez-vous
Écrivez 3 phrases pour vous présenter : Je m'appelle ___. Je suis ___ (nationalité). Je suis ___ (profession / étudiant·e). Je suis ___ (adjectif : content·e, fatigué·e, prêt·e...).
✏️ Exercice 3 — Formes négative et interrogative
Transformez les phrases :
1. Il est professeur. → Il professeur.
2. Vous êtes française. → Est-ce que vous française ?
3. Je suis fatigué. → Je fatigué.
4. Elle est étudiante. → Est-ce qu'elle étudiante ?
