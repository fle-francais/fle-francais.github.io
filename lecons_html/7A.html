Qu'est-ce que vous cherchez ?
Les vêtements, les démonstratifs, la couleur
💬 Dialogue
VENDEUR
Bonjour ! Qu'est-ce que vous cherchez ?
CLÉMENT
Je cherche une veste. Pour un mariage.
VENDEUR
Vous avez regardé nos nouvelles collections ? Regardez cette veste bleue.
CLÉMENT
C'est joli, mais c'est un peu trop formel. Et cette veste grise, là ?
VENDEUR
Très bon choix ! C'est élégant mais moderne. Vous faites quelle taille ?
CLÉMENT
Du 42, en général.
VENDEUR
Voilà un 42. Et cet imperméable, vous aimez ? Il va très bien avec.
CLÉMENT
Ces manteaux, là, c'est en soldes ?
VENDEUR
Oui, 30% de réduction. Vous voulez essayer la veste ?
CLÉMENT
Oui, je vais essayer ce modèle gris.
🔍 Compréhension
Pourquoi Clément cherche-t-il une veste ?
Pour un mariage.
Quelle est sa taille ?
Le 42.

📖 Point de langue — Les adjectifs démonstratifs — ce / cet / cette / ces
§§ Ce devant masculin. Cet devant masculin + voyelle/h muet. Cette devant féminin. Ces devant pluriel.
• Cette robe est magnifique. / Ces chaussures sont confortables.
• On peut ajouter -ci / -là : cette veste-ci (ici) / cette veste-là (là-bas).
• Forme négative : Ce pantalon n'est pas trop court. / Ces chaussures ne sont
• Forme interrogative : Est-ce que cette robe est en soldes ? / Ces chaussures
✏️ Exercice 1 — Ce, cet, cette ou ces ?
1. pantalon te va très bien.
2. chaussures sont à 40€.
3. imperméable est imperméable au vent.
4. robe est trop courte.
5. vêtements sont en soldes.
✏️ Exercice 2 — Jeu de rôle : en boutique
À deux : un vendeur, un client. Le client cherche : un pull pour l'hiver. Le vendeur propose différents modèles avec les démonstratifs : Ce pull…, Cette couleur…, Ces manches… Négociez la taille, la couleur, le prix.
✏️ Exercice 3 — Formes négative et interrogative
Transformez les phrases :
1. Cette veste est en soldes. → Forme négative :
2. Ces chaussures sont confortables. → Forme négative :
3. Ce manteau coûte 80€. → Forme interrogative (est-ce que) :
4. Cet article est disponible en bleu. → Forme interrogative (inversion) :
✏️ Exercice 4 — Vocabulaire : les vêtements et les achats
Complétez avec le mot correct : taille / soldes / essayer / élégant / réduction
1. Je voudrais cette veste, s'il vous plaît.
2. Je fais du 38. Vous avez cette robe en 38 ?
3. Ces chaussures sont très . Je les porte pour les soirées.
4. Il y a une de 20% sur tous les manteaux.
5. Les commencent en janvier et en juillet.
